# Sistema Inteligente de Gestión de Residuos Sólidos

## Descripción del proyecto

Este proyecto corresponde al desarrollo de un prototipo funcional orientado a optimizar la gestión de residuos sólidos en el distrito de Santa Marta mediante el uso de tecnologías de información.

El sistema fue desarrollado en Python utilizando una interfaz gráfica con Tkinter y una base de datos SQLite, permitiendo registrar rutas de recolección, almacenar información de residuos y generar reportes básicos para apoyar la toma de decisiones.

---

## Objetivo del proyecto

Diseñar y desarrollar un sistema de información en Python para optimizar el registro, control y análisis de datos relacionados con la gestión de residuos sólidos en Santa Marta.

---

## Tecnologías utilizadas

- Python
- Tkinter
- SQLite
- Visual Studio Code
- GitHub

---

## Funcionalidades principales

- Registro de rutas de recolección
- Registro de residuos sólidos
- Visualización de registros
- Eliminación de registros
- Dashboard con estadísticas
- Validación de datos
- Base de datos local SQLite

El desarrollo del sistema se realizó bajo un enfoque ágil utilizando la metodología Scrum, organizando el trabajo mediante fases iterativas de análisis, desarrollo, pruebas y validación.
